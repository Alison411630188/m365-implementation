# M365 導入專案網站 - 設計思路

## 選定設計方向：現代企業知識庫風格

我選擇了**現代企業知識庫風格**作為本網站的設計基調。這種風格強調清晰的資訊架構、專業的視覺層級，以及高效的內容導航。

### 設計理念

**設計運動**：企業級知識管理系統 + 現代極簡主義
- 參考 Microsoft 官方文檔風格、Notion 知識庫、以及現代 SaaS 平台的設計語言

**核心原則**：
1. **資訊優先**：清晰的內容層級，讓使用者快速找到所需資訊
2. **視覺專業性**：使用企業藍色、中性灰色，營造信任感與專業感
3. **高效導航**：側邊欄持久化導航，麵包屑路徑，快速搜尋功能
4. **可掃描性**：大量使用卡片、圖示、顏色標籤，讓內容易於掃描

**色彩哲學**：
- **主色**：Microsoft 365 藍色（#0078D4）- 代表信任、專業、創新
- **輔助色**：企業灰色系（#F3F2F1、#E1DFDD）- 提供視覺呼吸空間
- **強調色**：綠色（成功）、橙色（警告）、紅色（錯誤）- 用於狀態指示
- **文字**：深灰色（#323130）- 高對比度，易於閱讀

**佈局範式**：
- 側邊欄導航 + 主內容區域的經典企業佈局
- 卡片式內容分組，每個工具/主題獨立卡片
- 網格系統用於首頁特色內容展示
- 響應式設計：平板上側邊欄可收起，手機上轉為頂部導航

**簽名元素**：
1. **藍色強調線**：在卡片、標題、導航項目上使用左邊框或上邊框
2. **圖示系統**：每個 M365 工具配備獨特圖示（List、Loop、Planner 等）
3. **漸進式揭示**：展開/收起動畫，層級明確的內容摺疊

**互動哲學**：
- 平滑的過渡動畫（200-300ms）
- 清晰的懸停狀態（背景色變化、陰影提升）
- 點擊回饋（按鈕按下效果、載入動畫）
- 導航狀態視覺反饋（當前頁面高亮）

**動畫指南**：
- 頁面進入：淡入 + 輕微向上滑動（300ms）
- 卡片懸停：陰影增強、背景微妙變色（200ms）
- 側邊欄展開/收起：平滑寬度變化（250ms）
- 內容摺疊：高度動畫 + 淡入淡出（300ms）

**字體系統**：
- **標題字體**：Noto Sans TC（700 weight）- 現代、清晰、支援繁體中文
- **正文字體**：Noto Sans TC（400 weight）- 高可讀性
- **代碼字體**：Fira Code（用於技術內容）
- **層級**：
  - H1: 32px, 700 weight
  - H2: 24px, 700 weight
  - H3: 18px, 600 weight
  - 正文: 14px, 400 weight
  - 小字: 12px, 400 weight
